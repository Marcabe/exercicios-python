# Exercícios de Python

Este repositório reúne exercícios desenvolvidos durante os meus estudos de Python e lógica de programação.

O objetivo é praticar os fundamentos da linguagem, melhorar a resolução de problemas e acompanhar a minha evolução como estudante de Programação Informática.

## Conteúdos praticados

- Variáveis e tipos de dados
- Condições
- Ciclos de repetição
- Funções
- Listas
- Dicionários
- Manipulação de ficheiros
- Organização de código
- Resolução de problemas com Python

## Objetivo

Utilizar este repositório como registo da minha evolução e como apoio ao desenvolvimento de projetos práticos.
